How to export data from ccusage

Run these commands locally, then upload the resulting JSON files below — or use bin/refresh.sh if you cloned the cc-dash repo:

npx ccusage@latest daily --json > daily.json
npx ccusage@latest monthly --json > monthly.json
npx ccusage@latest session --json > session.json

Everything runs in your browser. No data leaves your machine. Files are also cached in localStorage for convenience.
